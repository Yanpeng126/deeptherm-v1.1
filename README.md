# DeepTherm V1.1

An independent implementation of **DeepTherm** (Wang et al., *Energy and AI* 24, 2026, [doi:10.1016/j.egyai.2026.100677](https://doi.org/10.1016/j.egyai.2026.100677)), a unified deep learning model for predicting thermochemistry properties of gas-phase molecules.

This implementation is rewritten from the paper text on top of [chemprop v2.2.3](https://github.com/chemprop/chemprop). It is not affiliated with the original authors. The original code is at https://github.com/tairan-w/DeepTherm.

## Results

10-model ensemble test MAE on the 4,997-species curated dataset.

**Random splits** (compare against Fig. 6a-c of the paper):

| Target | This work | Paper (full DeepTherm) |
| --- | --- | --- |
| ΔHf,298K (kcal/mol) | 2.38 | 1.25 |
| S298K (cal/mol/K) | 1.49 | 1.31 |
| Cp,300K (cal/mol/K) | 0.76 | 0.86 |
| Cp,400K (cal/mol/K) | 0.78 | 0.82 |
| Cp,500K (cal/mol/K) | 0.75 | 0.87 |
| Cp,600K (cal/mol/K) | 0.72 | 0.84 |
| Cp,800K (cal/mol/K) | 0.70 | 0.92 |
| Cp,1000K (cal/mol/K) | 0.69 | 0.84 |
| Cp,1500K (cal/mol/K) | 0.63 | 0.82 |

Eight of nine random-split targets reach or exceed the paper. The residual ΔHf gap reflects the absence of QM9 transfer learning together with implementation details that the paper does not fully specify.

## Installation

Clone this repository.

Using conda:

```bash
cd deeptherm-v1.1
conda env create -f environment.yml
conda activate deeptherm
```

A100 or any single GPU with >=24 GB memory is recommended for full ensemble runs.

## Data preparation

Place the supplementary curated dataset (Supplementary Data 1 from the paper) as `data/raw/curated_dataset.xlsx`, then:

```bash
python src/prepare_data.py
```

This writes a canonicalized CSV to `data/curated_4997.csv` containing 4,997 species with 9 thermochemistry targets.

## Training a single model

```bash
python src/train.py \
    --csv data/curated_4997.csv \
    --save-dir runs/single \
    --epochs 300 \
    --batch-size 32 \
    --d-hidden 600 \
    --depth 5 \
    --ecfp-bits 1024 \
    --seed 42
```

Per-target test MAE is printed to stdout and a `predictions.npz` is written to the save directory.

## Running the 10-model ensemble

Hyperparameters in `run_ensemble.sh` are set to the values that worked best in our grid search.

**Random split**:
```bash
SPLIT_MODE=random OUT_ROOT=runs/ensemble_random bash run_ensemble.sh
```

## File layout

```
deeptherm-v1.1/
├── data/
│   ├── raw/curated_dataset.xlsx     # user-supplied
│   └── curated_4997.csv             # generated by prepare_data.py
├── src/
│   ├── prepare_data.py              # xlsx -> canonicalized csv
│   ├── dataset.py                   # data loading and splits
│   ├── model.py                     # encoder, attention, ECFP projection
│   ├── train.py                     # single-seed training
│   ├── ensemble.py                  # inverse-MAE weighted aggregation
│   └── grid_aggregate.py            # grid search summarizer
├── run_ensemble.sh                  # 10-seed ensemble driver
├── grid_search.sh                   # hyperparameter grid driver
└── README.md
```
